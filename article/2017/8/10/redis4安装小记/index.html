    • REDIS4安装小记

    安装

    下载，解压，编译:

    $ wget http://download.redis.io/releases/redis-4.0.1.tar.gz
$ tar xzf redis-4.0.1.tar.gz
$ cd redis-4.0.1
$ make

    Make后查看日志提示

    make[3]: gcc: Command not found

    新机器没有安装gcc环境，因此在进行编译时才会出现上面提示

    安装gcc

    yum -y install gcc automake autoconf libtool make

    安装完GCC后，再次Make文件缺失，建议重新解压编译，安装成功

    二进制文件是编译完成后在src目录下，配置文件在根目录，按需移动

    通过下面的命令启动Redis服务：

    $ /usr/local/redis-4.0.1/bin/redis-server /usr/local/redis-4.0.1/etc/redis.conf

    你可以使用内置的客户端命令redis-cli进行测试：

    $ /usr/local/redis-4.0.1/bin/redis-cli -v
redis-cli 4.0.1
